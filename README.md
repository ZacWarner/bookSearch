# bookSearch
Find books and save them to read later!

## About
the search page lets you use googles book api to search for books, you can then save them to your list on the saved page.

## Deployed link
https://zacsbooksearch.herokuapp.com/

##Technologies
* React
* express
* react router
* ajax
* jsx
* Javascript
* Css
